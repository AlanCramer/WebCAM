What is CNC?

Maybe in some sense it is the opposite of 3D Printing. Instead of adding material layer by layer, it cuts out material from a sheet. Maybe carving something intricate or carving parts that can be assembled into something else.

How does this page work?

User draws a picture of what they want, enters information about the CNC machine they will use, and downloads an instruction file to feed to the machine.

About Resolutions

There are several related concepts about resolutions.

  • Import Image Resolution
    Imported images have a width and height. PNG and SVG might map these to physical units.
  • Canvas Model Resolution
    When the canvas is created, there is a width and height. This describes the canvas model.
  • Canvas Display Resolution
    The canvas can be styled and given a new width and height. This changes how the canvas gets displayed but not the underlying model.
  • Toolpath Calculation Resoution
    To calculate the toolpaths, the canvas will be copied to another image which is not displayed. This image can have yet another resolution.
